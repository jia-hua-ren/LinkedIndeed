# LinkedIndeed
Are you tracking your job apps with an excel / google Sheets spreadsheet? This extension is for you! 

# How to install
This extension is officially on the Chrome Web Store! Install here: https://chromewebstore.google.com/detail/linked-indeed/mkjfhangliapdjidllfldlgcjdccfaoc?authuser=0&hl=en

To load unpacked: Download the repo as a zip -> extract -> go to chrome://extensions -> turn on Developer Mode -> load unpacked -> select the unzipped folder.

# How to use
First, open up the extension and go to the "settings" tab. This is where you will set the column formats specific to your excel job tracker. If your specific column type is missing, you can suggest it, but for now you can leave it as blank so that it all aligns (very important! If you don't leave space for these, the copy-paste won't be aligned). 

Once you have set the settings, when you open up a specific job post on a separate tab (important!) from a supported job platform (currently Indeed, LinkedIn, and Greenhouse), click the extension and you will see some information at a glance, and click "Copy Excel" to copy the row. You can now paste the information into your personal Excel sheet, no more back-and-forth copy-pasting!!
